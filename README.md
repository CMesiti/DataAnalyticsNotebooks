# DataAnalyticsNotebooks
Collection of notebooks performing common analytics tasks with a variety of data sets, data types, and varying sizes with data visualizations and insights driven by data. This repository is a resource for practice and examples on basic data preprocessing for machine learning tasks and driving business decisions.

## Basic NLP
- Within the TextAnalysis notebook we explore basic NLP techniques such as the following:
- EDA with Text
- Tokenization with NLTK
- Frequency Analysis
- Semantic Classification & Inference

## Data Challenge
- Within the Claims Data Challenge notebook we explore utilizing a range of techniques to pull insights and answer questions about the data
- EDA
- Correlations and Relationships
- Summarizing and Visualizing Data
- Text analysis including Clustering, Token Frequency, TF-IDF, and LDA Topic Modeling.
